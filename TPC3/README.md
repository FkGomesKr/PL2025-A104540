Este trabalho prático é um pequeno conversor de **MarkDown** para **HTML** feito em **Python**. Este conversor está aplicado a alguns elementos básicos:
- Cabeçalhos: linhas iniciadas por "# texto", ou "## texto" ou "### texto"
   In: # Exemplo
